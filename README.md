# Velonutri

A Progressive Web App for calculating carbohydrate needs for cycling and generating gel flask recipes.

## Features

- Calculate total carbs based on ride duration and intensity
- Preset intensity levels (Easy/Hard) or custom carbohydrate rate input
- Gel flask recipe generator with configurable concentration
- Optional maltodextrin-fructose mix ratio
- Works offline as a PWA
- Fully responsive design

## Tech Stack

- Plain JavaScript (no dependencies)
- PWA with Service Worker
- HTML/CSS

## Usage

1. Select your planned ride duration using the +/- buttons
2. Choose intensity level (Easy, Hard, or Custom)
3. View total carbohydrates needed and gel flask recipe

## Hosting

### Online
Visit: https://yanniklubas.github.io/velonutri/

### Local Development
```bash
cd velonutri
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Installation

To install as a PWA on mobile:
- **iOS**: Open in Safari → Share → Add to Home Screen
- **Android**: Open in Chrome → Install App (or menu → Add to Home Screen)
